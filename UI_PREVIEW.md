## Key Features Illustrated

### Campaign Independence
- Campaign 1: Instagram + TikTok, uploads every 2h 30m
- Campaign 2: YouTube only, uploads every 1h
- Campaign 3: All platforms, manual uploads (scheduling disabled)

Each operates independently without affecting others.

### Video Flexibility
The same video can be in multiple campaigns with different metadata:
- Campaign 1: "Beach Sunset - Summer Promo" + #summer #sale
- Campaign 2: "Relaxing Beach Vibes" + #meditation #calm
- Campaign 3: "Vacation Inspiration" + #travel #explore

### Workflow
1. Create campaign(s) with desired platforms and schedules
2. Add videos from registry to campaign(s)
3. Customize metadata for each video in each campaign
4. Start campaign scheduler
5. Videos upload automatically according to each campaign's schedule
